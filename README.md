# FERMÀT Account Management Portal

A visual, data-driven dashboard for CSMs at FERMÀT to manage their brand portfolio. Built as a static HTML file generated by Python (`build.py`), powered by real data from FERMÀT Platform, Slack, Gmail, Glyphic, Granola, and Google Calendar MCPs.

---

## Features

- **Portfolio Overview** — bird's-eye view of all 15 brands with health scores, revenue sparklines, and status indicators
- **Health Scoring** — 3-factor rubric (Platform Performance /35, Communication /30, Engagement /35) computed from real MCP data
- **Performance Tab** — date range selector (Yesterday to 45 Day), daily charts with data labels, conversion funnel with real pdp_views/atc/checkouts, executive summary
- **Calls Timeline** — visual timeline from Glyphic + Granola
- **Slack Chat Bubbles** — channel messages in chat UI
- **Email Inbox** — Gmail threads in inbox design
- **Calendar View** — upcoming/past events

---

## Architecture

```
account-management-portal/
├── build.py              # Main builder — generates index.html from data + JS/CSS
├── compute_health.py     # 3-factor health score calculator
├── index.html            # Generated dashboard (open in browser)
├── org_id_map.json       # Maps CSV brand IDs to FERMÀT org IDs
├── data/                 # Per-brand JSON data files (15 brands)
│   ├── {brand_id}.json   # Health, performance, calls, slack, email, calendar
│   └── ...
├── js/                   # Legacy JS files (unused — build.py embeds JS)
├── css/                  # Legacy CSS files (unused — build.py embeds CSS)
└── update_*.py           # Data update scripts
```

## How to Use

### Quick Start

```bash
open index.html  # View the dashboard
```

### Refresh All Data (full pipeline)

Use the Claude Code skill:

```
/account-dashboard
```

This runs the complete pipeline:

1. Pulls FERMÀT funnel metrics (sessions, revenue, CVR, purchases, pdp_views, atc, checkouts + daily arrays)
2. Pulls Slack messages, Gmail emails, Glyphic calls, Calendar events
3. Writes JSON data files
4. Runs `compute_health.py` (3-factor health scores)
5. Runs `build.py` (generates index.html)
6. Opens in browser

### Manual Steps

```bash
# Just recompute health scores from existing data
python3 compute_health.py

# Just rebuild HTML from existing data
python3 build.py

# Full pipeline
python3 compute_health.py && python3 build.py && open index.html
```

## Health Score Rubric

### Platform Performance (35 pts)

4 metrics from FERMÀT API:

| Metric | 9 pts | 7 pts | 5 pts | 3 pts | 2 pts | 0 pts |
|--------|-------|-------|-------|-------|-------|-------|
| 7d Sessions | >1,000 | >500 | >200 | >50 | — | 0 |
| 7d Revenue | >$5,000 | >$1,000 | >$200 | — | >$0 | $0 |
| 7d CVR | >2% | >1% | >0.5% | — | >0% | 0% |

| Metric | 8 pts | 6 pts | 4 pts | 2 pts | 0 pts |
|--------|-------|-------|-------|-------|-------|
| 7d Purchases | >30 | >10 | >5 | >0 | 0 |

### Communication (30 pts)

5 metrics from Slack + Gmail:

- Slack recency
- Brand Slack activity
- Pulkit Slack activity
- Email threads
- Outreach sent

### Engagement (35 pts)

3 metrics from Glyphic + Granola + Calendar:

- Last call date: /12
- Calls in 30d: /12
- Upcoming call scheduled: /11

### Thresholds

| Score | Color |
|-------|-------|
| 70+ | Green |
| 40-69 | Yellow |
| 0-39 | Red |

## Data Sources

| Source | MCP | What it provides |
|--------|-----|-----------------|
| FERMÀT Platform | `get_funnel_metrics` | Sessions, revenue, CVR, purchases, pdp_views, atc, checkouts, daily data |
| Slack | `slack_read_channel` | Channel messages, activity analysis |
| Gmail | `gmail_search_messages` | Email threads, outreach tracking |
| Glyphic | `list_calls`, `get_call_info` | Call recordings, summaries, action items |
| Granola | `query_granola_meetings` | Meeting notes, transcripts |
| Google Calendar | `gcal_list_events` | Upcoming/past events |

## Key Rules

- **ZERO hallucination** — every number comes from real MCP API responses
- **No approximations** — pdp_views, atc, checkouts are real, never estimated
- Full numbers with commas ($83,401 not $83.4K)
- FERMÀT (with accent) — never "Fermat"
